- The ratio of END eigenvalue to START eigenvalue IS the **Terminal Concentration Factor**.

## IV. THE CLAIMS

### C186 — Terminal Concentration Theorem

**Statement:** For any truck route modeled as a directed linear chain with terminal self-loops, the END nodes have a higher eigenvector centrality than START nodes. The ratio (Terminal Concentration Factor) is always > 1.0. This means END cities concentrate evidence (bodies, drugs, trafficked persons, illicit cargo) at a rate proportional to the terminal concentration factor. The body is at the terminus.

**Falsification:** If any truck route shows terminal concentration < 1.0 (END cities disperse more than START cities), the theorem is false. All 10 tested routes show > 1.0.

### C187 — Dispersal-Asymmetry Principle

**Statement:** The asymmetry between START (dispersal) and END (accumulation) is structural, not statistical. START nodes lack self-loops because cargo departs without returning. END nodes have self-loops because cargo arrives and remains. The asymmetry IS the trafficking route. Every trafficking route has this asymmetry. The asymmetry IS the evidence.

**Falsification:** If any trafficking route shows symmetric dispersal (equal self-loops at START and END), the asymmetry principle is false.

### C188 — Missing Persons Terminus Corollary

**Statement:** Missing persons from BEGINNING cities are more likely to be found at END cities than at other BEGINNING cities or MIDDLE cities. The probability of finding a missing person at an END city is proportional to the Terminal Concentration Factor of the route connecting their disappearance point to that END city. For I-80 (concentration 3.122), a missing person from New York is 3.1x more likely to be found in San Francisco/Oakland/Sacramento/Reno than random chance would predict.

**Falsification:** If missing persons from START cities are found at other START cities or MIDDLE cities at rates exceeding END cities, the corollary is false.
